# PlayPath Roblox SDK Docs

This repository is intended to be usable by:

- Roblox studios integrating PlayPath into production games
- teams evaluating the SDK with the bundled Math/ELA demo
- agents and internal tools that need stable, task-oriented entry points

## Start Here

- New to the SDK: [ModuleScript Quickstart](./guides/module-script-quickstart.md)
- Shipping with Rojo/Wally: [Rojo + Wally Quickstart](./guides/rojo-wally-quickstart.md)
- Building a real game loop: [Integration Guide](./guides/integration-guide.md)
- Evaluating the sample game: [Math + ELA Demo Guide](./guides/demo-guide.md)
- Copying a production-ready wrapper shape: [`examples/production`](../examples/production/README.md)
- Getting launch-ready: [Launch Checklist](./guides/launch-checklist.md)
- Diagnosing integration issues: [Troubleshooting](./guides/troubleshooting.md)
- Raw HTTP/API contract: [`ROBLOX_SDK_API.md`](../ROBLOX_SDK_API.md)

## Recommended Reading Order

For a first integration:
1. Read one installation quickstart.
2. Read the integration guide.
3. Read the launch checklist.

For agent-assisted workflows:
1. Read one installation quickstart.
2. Read the integration guide.
3. Use the API contract for method and payload details.

For demo evaluation:
1. Read the demo guide.
2. Run the smoke checklist in mock mode.
3. Repeat against beta with real credentials.
